    Control Blocks

    Control Blocks are for composing an HTML form with Compute layout block.

    Arakawa supports various control blocks.

    ar.TextBox("text")

    ar.NumberBox("number")

    ar.RangeField(name="range", min_=0, max_=100, step=10)

    ar.SwitchField("switch")

    ar.FileField("file")
